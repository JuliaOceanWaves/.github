# JuliaOceanWaves

This repo creates [default community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file) for repos in the _JuliaOceanWaves_ organization. Repos should rely on these files unless there is a compelling reason for bespoke community health files. To use these files simply do not create the corresponding files in the repo. Repos in the _JuliaOceanWaves_ organization that do not define their own will automatically inherit these.
